# 신협 공제 상품 (연계 추천용)

## 대출금상환공제 ⭐ (여신 고객 핵심)

**무엇인가**
- 가입자가 사망·중대질병 시 **잔여 대출금을 공제가 대신 상환**
- 남은 가족에게 주담대 부담이 전가되지 않음

**누구에게 필수인가**
- **가장 역할인 40대·50대** (자녀 양육기)
- **부양가족 있는 주담대 보유자**
- **외벌이·맞벌이 모두 해당**

**CFP 관점 추천 이유**
- 생명보험의 "잔여대출 상환" 기능을 단독 상품으로 분리한 형태
- 주담대와 자연스럽게 연계되는 보장 설계
- 종신보험보다 저렴한 경우가 많음 (잔액에 연동되므로 시간 지날수록 보장 감소 = 보험료 낮음)

## 화재공제

**무엇인가**
- 담보 주택의 화재·풍수해 손해 보상
- 주담대 담보가치 보존 차원에서 연계 권장

**누구에게 필요한가**
- **주담대 보유자 (사실상 전원 권장)**
- 자가 소유자
- 전세·월세는 가재도구·생활용품 중심으로 별도 설계

**CFP 관점**
- 담보 물건이 소실되면 대출은 남는다 → 리스크 필수 대비
- 공제료 대비 보장 범위가 큼 (저비용 고효과)

## 종신공제

**무엇인가**
- 평생 보장되는 생명보험 형태의 공제
- 사망 시 지정 수익자에게 공제금 지급

**누구에게 추천**
- **50대 이상 상속 계획 시작 시점**
- 자산 이전·상속세 준비
- 이미 대출금상환공제가 있는 경우 추가 고려

**주의**
- 40대 자녀 양육기에는 **대출금상환공제가 우선**
- 종신공제는 여유 자금이 생긴 후 (은퇴 5년 전쯤) 검토 권장

## 공제와 보험의 차이

- 공제 = 상호부조 원리, 조합원 기반
- 보험 = 영리보험사 상품
- 공제는 **배당 가능**, 낮은 사업비, 조합원 우대
- 보장 내용·약관은 상품별로 다름 — 반드시 상담 시 확인
