# 대출 관리 전략 (CFP 기반)

## 주택담보대출 — 30년 여정

주담대는 가장 긴 금융상품이다. 계약 당시 조건이 "최종 조건"이 아니다.
계약 후에도 여러 지렛대가 있음:

### 1. 금리인하요구권 (가장 강력한 무료 지렛대)

**언제 신청 가능한가?**
- 취업 / 승진 / 연봉 인상
- 소득 증빙 개선 (1년치 원천징수영수증 확보)
- 신용점수 상승
- 부채 상환으로 DSR 개선
- 전문자격 취득
- 타 대출 상환 완료

**효과**
- 평균 0.2~0.5%p 인하 성공 사례 많음
- 0.3%p만 인하돼도 2.5억 30년 기준 약 1,500만원 총 이자 절감

**실행 팁**
- 연 1회는 점검. 신청은 신협 담당자에게 문의
- 거절되어도 불이익 없음 (패널티 없음)
- 거절 이유 들어두면 다음 시도에 유리

### 2. 중도상환 — "커피 한 잔의 복리"

**효과 (2.5억 / 30년 / 4.2% 기준)**
- 매달 5만원 추가 → 약 2.2년 단축, 1,600만원 절감
- 매달 10만원 추가 → 약 4.1년 단축, 3,000만원 절감
- 매달 30만원 추가 → 약 9.6년 단축, 6,800만원 절감

**전략**
- 초기 상환 효과가 가장 큼 (초기에 이자 비중이 크기 때문)
- 중도상환 수수료 확인 필수 (보통 3년 이내는 수수료 있음)
- 비상자금 6개월치 확보한 후에만 시작할 것

### 3. 상환 전략 선택

**원리금균등 (가장 흔함)**
- 매달 동일한 금액 → 가계부 관리 쉬움
- 초기 이자 비중 큼

**원금균등**
- 초기 부담 크지만 총 이자 적음
- 소득이 감소할 예정인 경우 비추천 (후반에 덜 내는 게 좋으니)

**만기일시**
- 주담대에선 거의 안 씀
- 사업자 대출 단기에만

## 대출 고객의 수신 전략 (교차 전략)

**핵심 원칙:** *"대출 있으면 저축 못 한다"는 거짓말.*

### 비상자금 우선 (6개월치 생활비)
- 대출이자보다 낮은 수신금리여도 **반드시 분리 보관**
- "있어야 안심되는 돈"
- 추천 상품: 신협 자유적립예금

### 조합원 우대 예금
- 신협 대출 고객 대상 우대금리 적용 상품 존재
- 타 은행 적금 대비 0.2~0.5%p 우위 가능
- 상담 시 담당자에게 문의 권장

### 자녀 교육·주택 이사 자금은 별도 적립
- 대출 상환과 별개 트랙
- 중기 목적자금은 청약·정기적금 병행

## 신협 특유의 관계형 가치

- **담당자 직접 상담** — 약정서에 없는 상황별 유연성 (연체 초기·대환 등)
- **조합원 혜택** — 가입 시 출자금, 배당·우대 가능
- **예금자보호** — 최대 5,000만원까지 보호 (시중은행과 동일)
- **연체 시 기계적 독촉 대신 사람 먼저** — 어려울 때 상의 가능
- **지역 기반** — 오프라인 지점 접근성
