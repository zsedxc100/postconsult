# 백로그

> 1주차 MVP에 들어가지 않는 아이디어들. 1주차가 끝나면 여기서 다시 골라.

## 2주차 후보
- A 케이스 (수신 상담만)
- B 케이스 (수신 가입 완료)
- C 케이스 (여신 약정 전)
- 칩 라이브러리 30~40개 정의
- DISC × BIT 8가지 조합 톤 검증
- 알림톡 실 발송 (Aligo 또는 본점 엠피인터랙티브)

## 추후
- 건보료 계산기 (Tool Use)
- 세법 시뮬레이션
- 본인 카톡으로 보내기 (카카오 공유 SDK)
- QR 코드 미리보기
- 본점 통합용 발송 모듈 교체
- 분기별 KB 자동 갱신 알림
- 사내 SSO 연동
- 직원 사용 통계 대시보드
- A/B 톤 비교 데모 페이지
